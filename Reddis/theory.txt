#Redis Uses Its Own Protocol

Redis uses a protocol called RESP (Redis Serialization Protocol).
When you write:

await redis.set("name", "Abhi");

ioredis converts it into something like:

*3
$3
SET
$4
name
$4
Abhi

and sends it directly to Redis over a TCP connection.

MongoDB Uses Its Own Wire Protocol

Similarly, when you write:

await User.find();

Mongoose converts that into MongoDB's binary wire protocol.

MongoDB receives:

find users collection

(in binary format, not HTTP)

and returns the documents.
